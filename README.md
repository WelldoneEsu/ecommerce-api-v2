# Project Title
- ecommerce-api-v2

## Description
- A RESTful API for eCommerce applications, providing endpoints for managing products, orders, customers, and payments.


## Features
- Authentication 
- CRUD operations 
- MongoDB Integration
- Product management
- Order management
- Customer management
- Payment processing



## Installation
- 
git clone https://welldoneEsu/ecommerce-api-v2.git
cd project-folder
npm install

 ## Usage 
 npm start


## Technologies 
- Node.js
- Express.js
- MongoDB
